Computer MCQs

  • 1. What does CPU stand for in computer terminology?

    • A. Central Process Unit
    • B. Central Processing Unit
    • C. Central Print Unit
    • D. Central Processing Utility

    Answer: B

  • 2. What is the primary function of RAM in a computer?

    • A. To store permanent data
    • B. To process graphics
    • C. To run applications and store temporary data
    • D. To manage network connections

    Answer: C

  • 3. Which one of the following is a programming language?

    • A. HTTP
    • B. HTML
    • C. FTP
    • D. Java

    Answer: D

  • 4. What does 'GUI' stand for?

    • A. General User Interaction
    • B. Graphical User Interface
    • C. Graphical User Interaction
    • D. General User Interface

    Answer: B

  • 5. What is the term for a network that connects computers within a limited area such as a residence or school?

    • A. WAN
    • B. MAN
    • C. LAN
    • D. PAN

    Answer: C

  • 6. What does 'HTTP' stand for?

    • A. HyperText Transfer Protocol
    • B. HyperText Transfer Platform
    • C. HyperText Transmission Protocol
    • D. HyperText Transmission Platform

    Answer: A

  • 7. Which device is primarily used for providing hard copy of digital documents?

    • A. Monitor
    • B. Printer
    • C. Scanner
    • D. Keyboard

    Answer: B

  • 8. What does 'OS' stand for in computer terminology?

    • A. Open Source
    • B. Operating System
    • C. Optical System
    • D. Online System

    Answer: B

  • 9. Which of the following is a popular version control system?

    • A. Git
    • B. Bit
    • C. Dot
    • D. Pit

    Answer: A

  • 10. What is the full form of 'SSD' in computer storage?

    • A. Solid State Disk
    • B. Solid State Drive
    • C. Secure Storage Device
    • D. Secure State Disk

    Answer: B
